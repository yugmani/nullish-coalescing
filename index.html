What is Nullish Coalescing(or ??) in JavaScript?

  • In JavaScript, the nullish coalescing operator, or ?? operator is used to return the right hand side whenever the left hand side is null or undefined.
  • The nullish coalescing operator is useful in situations where something can be returned as either null or undefined, and helps us tighten up our code. For example, a function that returns undefined in some situations can be provided with a default value.

Differences between the logical OR operator and Nullish Coalescing operator

  • Falsy Values

    1. false
    2. 0 or -0 or 0n
    3. empty string i.e. ""
    4. null
    5. undefined
    6. NaN
  • In the past, We typically set default values in JavaScript using the logical OR || operator.
  • The || operator returns the right hand value if the left hand value is falsy(any of six falsy value).
  • In the other hand, the nullish coalescing operator ?? triggers if a value is null or undefined. Because, it considers other falsy values a valid values.

Chaining with nullish coalescing operator

  • It is also possible to chain the nullish coalescing operator.
  • But we cannot chain it with the logical || operator, unless with parenthesis.
